# 💼 Currículum Digital - Milciades Almirón González
 
Currículum web profesional, moderno e interactivo con opción de descargar PDF y contactar por WhatsApp.
 
## ✨ Características
 
- 🎨 **Diseño Moderno**: Interfaz profesional y atractiva
- 📥 **Descarga de PDF**: Genera un documento listo para imprimir
- 💬 **Contacto WhatsApp**: Abre WhatsApp automáticamente
- 📱 **Responsive**: Funciona perfectamente en todos los dispositivos
- ⚡ **Rápido**: Optimizado para máximo rendimiento
- 🌐 **Compartible**: Enlace único para compartir
- 🔍 **SEO Optimizado**: Descubrible en buscadores
 
## 🚀 Despliegue en Vercel
 
1. Clona este repositorio
2. Corre: `npm install`
3. Corre: `npm run dev` para probar localmente
4. Sube a GitHub: `git push origin main`
5. Ve a https://vercel.com y conecta tu repositorio
6. Vercel desplegará automáticamente
 
## 🛠️ Tecnologías
 
- Next.js 14
- React 18
- Tailwind CSS
- Lucide React
 
## 📝 Personalización
 
Edita `components/CurriculumWeb.jsx` para cambiar:
- Nombre
- Teléfono
- Correo
- Número de WhatsApp
- Contenido del CV
- Colores (en tailwind.config.js)
